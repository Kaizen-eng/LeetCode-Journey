# Notes

## Observation
The value at each position tells us the next index to access.

Example:

nums = [0,2,1,5,3,4]

ans[0] = nums[0]
ans[1] = nums[2]
ans[2] = nums[1]
...

## Key Learning

- Read the problem carefully because values are used as indices.
- Index manipulation becomes much easier after writing one or two examples.
- Simple one-pass traversal is enough.
